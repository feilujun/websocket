# 量投前端框架模板

----
## 目录结构
     |---config 
     |--- public
     |--- scripts
     |--- src
     |      |--- assets              #  图片等本地静态资源
     |      |--- common
     |          |--- config.js       #  title、logle等资源配置项
     |          |--- menu.js         #  菜单配置项
     |          |--- router.js       #  路由设置项
     |      |--- components          #  自定义的业务通用组件
     |      |--- language            #  多语言设置
     |      |--- layouts             #  通用布局
     |      |--- models              #  全局各个页面的modal
     |      |--- routes              #  各个业务页面详细信息
     |      |--- utils               #  工具库
     |      |--- index.js            #  项目入口
     |      |--- router.js           #  路由配置
     |--- .env.production
     |--- .gitignore
     |--- config-overrides.js
     |--- README.md
     |--- package.json           
     |--- yarn.lock
